# Examples

This directory contains fictional sample data for credential-free demos.

Do not place private resumes, API keys, OAuth tokens, or real job-search data
in this directory.

`source_packs/` contains public source configuration only. Default RSS entries
are intended to work for all users without credentials. Web and regulatory
entries are public metadata and health-check targets until dedicated parsers are
added.

Keep private watchlists, paid newsletters, raw `.eml` exports, and API-backed
source configs outside the repository.
